# AirPods Connector

Another Alfred AirPods Connector with zero configuration. Supports more than one pair of AirPods

**Usage**: `airpd`

## Supported headphones

- AirPods 1/2/3/4
- AirPods Pro 1/2
- AirPods Max
- Powerbeats Pro

## Required

* Python 3

## TIP

To show connection status on Desktop use Uebersicht, check it out:

1. Uebersicht: https://github.com/felixhageloh/uebersicht
2. Uebersicht-Widget: https://github.com/Acidham/uebersicht-airpod-status
